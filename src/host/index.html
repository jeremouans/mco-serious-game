Démarrer une session

  1. Choisissez un quiz dans l'onglet Sessions
  2. Cliquez "Démarrer une session"
  3. Partagez le code ou le QR code
  4. Attendez les joueurs → Démarrez

Questions quiz

Laissez les joueurs répondre. Cliquez "Révéler" quand vous êtes prêt — même avant le timer. Le classement s'affiche automatiquement.

Commentez la bonne réponse avant de passer à la suite.

🎚️

Slider (estimation)

Les joueurs estiment une valeur numérique. Le plus proche gagne. Cliquez "Révéler" pour voir les estimations animées.

Idéal pour des statistiques surprenantes !

☁️

Nuage de mots

Les joueurs choisissent des mots dans une liste. Le nuage se forme en direct. Cliquez "Afficher" pour le révéler.

Bon brise-glace ou conclusion de manche.

💡

L'idée en Or

Les joueurs soumettent une idée, puis votent pour leur préférée (anonymement). L'idée gagnante est révélée avec confettis.

Passez au vote quand assez d'idées sont soumises.

🤝

Mes engagements

Les joueurs rédigent un engagement. Les autres les notent de 1 à 5 étoiles sur 4 critères. Vous pouvez ajouter un bonus.

Clôturez avec les engagements les mieux notés.

📊

Scoring

  • Quiz / Vrai-Faux : jusqu'à 1 000 pts
  • Slider : rang 1 = 1 000 pts, rang 2 = 700…
  • Idée en Or : 100 pts/vote + 300 pts gagnant
  • Engagements : notation étoiles × 50 pts
